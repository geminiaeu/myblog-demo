::: tip 行路难
金樽清酒斗十千，玉盘珍羞直万钱。<br />
停杯投箸不能食，拔剑四顾心茫然。<br />
欲渡黄河冰塞川，将登太行雪满山。<br />
闲来垂钓碧溪上，忽复乘舟梦日边。<br />
行路难，行路难，多歧路，今安在？<br />
